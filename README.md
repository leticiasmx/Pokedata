# PokéData Pro

Uma Pokédex profissional e elegante que consome a PokeAPI para exibir informações detalhadas sobre Pokémon. Desenvolvida com design dark moderno, tradução automática e interface responsiva.

![PokéData Pro](https://img.shields.io/badge/version-1.0-d48aa3?style=flat-square)
![API](https://img.shields.io/badge/PokeAPI-v2-blue?style=flat-square)
![License](https://img.shields.io/badge/license-MIT-green?style=flat-square)

## ✨ Funcionalidades

- **Busca por nome ou número** - Encontre qualquer Pokémon rapidamente
- **Pokémon aleatório** - Botão surpresa para descobrir novas espécies
- **Design dark profissional** - Interface com tema escuro elegante
- **Tradução automática** - Nomes de tipos, estatísticas e descrições para português
- **Dados completos**:
  - ✨ Imagem oficial (artwork)
  - 📊 Estatísticas base com barras visuais
  - 🏷️ Tipos do Pokémon
  - 📏 Altura e peso
  - ⚡ Experiência base
  - 📖 Descrição da espécie (prioridade para português)

## 🚀 Como usar

1. Clone ou baixe o arquivo HTML
2. Abra diretamente no seu navegador (não requer servidor)
3. Digite o nome ou número de um Pokémon
4. Clique em "Buscar" ou pressione Enter
5. Use o botão 🎲 para um Pokémon aleatório

## 📦 API utilizada
PokeAPI - A RESTful API mais completa sobre Pokémon

### Endpoints consumidos:
GET /pokemon/{id} - Dados principais e estatísticas

GET /pokemon-species/{id} - Descrições e dados da espécie

## 🛠️ Tecnologias
HTML5 semântico

CSS3 moderno (Grid, Flexbox, Variáveis CSS)

JavaScript ES6+ (Async/Await, Fetch API)

PokeAPI v2
